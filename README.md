# ShopMart Website Documentation

This document provides an overview of the HTML, CSS, JavaScript, and Bootstrap components used in the ShopMart website.

## HTML Components

### 1. Navigation Bar
- **HTML File:** index.html
- **Description:** The navigation bar, which includes links to various sections of the website, such as Home, Contact, Products, and Login.

### 2. Welcome Message
- **HTML File:** index.html
- **Description:** A welcome message displayed on the home page.

### 3. Carousel
- **HTML File:** index.html
- **Description:** A carousel with images and captions showcasing different products and offers.

### 4. Accordion (Category Sections)
- **HTML File:** index.html
- **Description:** Accordion sections for different product categories. Each section includes a list of products.

### 5. Recommended Products
- **HTML File:** index.html
- **Description:** Displays recommended products with images, titles, and descriptions.

### 6. New Arrivals
- **HTML File:** index.html
- **Description:** Displays new arrival products with images, titles, descriptions, and a "New" badge.

### 7. Offer Alert
- **HTML File:** index.html
- **Description:** An alert that displays an exciting offer and a progress bar indicating the offer's popularity.

### 8. Team Details
- **HTML File:** index.html
- **Description:** A section that displays details about the team members.

### 9. Pagination
- **HTML File:** index.html
- **Description:** Navigation links for different webpages (Home, Login, Signup, Contact).

### 10. Jumbotron
- **HTML File:** index.html
- **Description:** A prominent Jumbotron component that may include a call to action or key information.

### 11. Login Form
- **HTML File:** index.html
- **Description:** A login form for users to enter their login credentials.

## CSS Components

### 1. Global Styling
- **CSS File:** styles.css
- **Description:** Includes global styling for the entire website, including margin and padding adjustments.

### 2. Navigation Bar Styling
- **CSS File:** styles.css
- **Description:** Defines styles for the navigation bar, including the color of links.

### 3. Custom Margin Styling
- **CSS File:** styles.css
- **Description:** Adds custom margins to various sections with box shadows.

### 4. Image Thumbnail Hover Effect
- **CSS File:** styles.css
- **Description:** Adds a hover effect to image thumbnails within cards.

### 5. Contact Form Styling
- **CSS File:** contact.css
- **Description:** Defines styles for the contact form, including width, border radius, and font styles.

### 6. Heading and Alert Styles
- **CSS File:** contact.css
- **Description:** Styles for headings, alert messages, and message input.

### 7. Send Button Styling
- **CSS File:** contact.css
- **Description:** Styles for the "Send" button, including border radius and gradient background.

## JavaScript Components

### 1. Offer Claim Button
- **JavaScript Code:** index.html
- **Description:** JavaScript code for handling the "Claim" button in the offer section, which displays an alert when clicked.

## Bootstrap Features and Components

### 1. Navigation Bar (Bootstrap Navbar)

- **File:** `index.html`, `login.html`, `signup.html`, `contact.html`
- **Section:** `<nav class="navbar">`
- **Description:** The navigation bar uses Bootstrap's navbar component to provide a responsive and customizable navigation menu.

### 2. Carousel (Bootstrap Carousel)

- **File:** `index.html`
- **Section:** `<div id="productCarousel" class="carousel slide custom-margin">`
- **Description:** The carousel is implemented using Bootstrap's Carousel component to display product images and captions in a sliding manner.

### 3. Accordion - Shop by Category (Bootstrap Collapse)

- **File:** `index.html`
- **Section:** `<div class="accordion" id="accordionExample">`
- **Description:** The accordion menu for exploring product categories is created using Bootstrap's Collapse component.

### 4. Alert (Bootstrap Alert)

- **File:** `contact.html`
- **Section:** `<div class="alert">`
- **Description:** Bootstrap's Alert component is used to display alert messages for the contact form.

### 5. Buttons (Bootstrap Buttons)

- **File:** `index.html`, `contact.html`, etc.
- **Section:** `<button class="btn">`
- **Description:** Bootstrap's button styles and classes are used for various buttons throughout the website, such as "Buy Now" and "Claim" buttons.

### 6. Progress Bar (Bootstrap Progress)

- **File:** `index.html`
- **Section:** `<div class="progress">`
- **Description:** Bootstrap's Progress component is utilized for displaying the progress of the offer alert.

### 7. Modal (Bootstrap Modal)

- **File:** `login.html`, `signup.html`
- **Section:** `<div class="modal">`
- **Description:** Bootstrap's Modal component is used to create login and signup modal dialogs.

### 8. Cards (Bootstrap Cards)

- **File:** `index.html`
- **Section:** `<div class="card">`
- **Description:** Bootstrap Cards are used for displaying product information and descriptions.

### 9. Badges (Bootstrap Badges)

- **File:** `index.html`, `signup.html`
- **Section:** `<span class="badge">`
- **Description:** Bootstrap Badges are used to highlight the "New" label for new arrivals.

### 10. Dropdown (Bootstrap Dropdown)

- **File:** `index.html`
- **Section:** `<li class="nav-item dropdown">`
- **Description:** Bootstrap Dropdowns are used for the Products menu.

### 11. Input Forms (Bootstrap Forms)

- **File:** `contact.html`
- **Section:** `<form class="form">`
- **Description:** Bootstrap Forms are used for the contact form.

### 12. Jumbotron (Bootstrap Jumbotron)

- **File:** `index.html`
- **Section:** `<div class="container mt-5 custom-container">`
- **Description:** Bootstrap Jumbotron is used for the welcome message.

### 13. Pagination (Bootstrap Pagination)

- **File:** `index.html`
- **Section:** `<nav aria-label="Product Pagination">`
- **Description:** Bootstrap Pagination is used for the webpage navigator.

### 14. Dropdown Toggle (Bootstrap Dropdown Toggle)

- **File:** `index.html`
- **Section:** `<a class="navbar-toggler" data-bs-toggle="collapse">`
- **Description:** Bootstrap Dropdown Toggle is used for the navigation bar toggle button.

### 15. Image Thumbnails (Bootstrap Thumbnails)

- **File:** `index.html`, `contact.html`
- **Section:** `<img class="img-thumbnail">`
- **Description:** Bootstrap Image Thumbnails are used for product images and thumbnails.

### 16. Popover (Bootstrap Popover)

- **File:** `index.html`
- **Section:** `<a tabindex="0" class="btn btn-link" data-toggle="popover">`
- **Description:** Bootstrap Popover is used for displaying offer details on the progress bar.

### 17. Dropdown Menu (Bootstrap Dropdown Menu)

- **File:** `index.html`
- **Section:** `<ul class="dropdown-menu">`
- **Description:** Bootstrap Dropdown Menu is used for the Products menu.
